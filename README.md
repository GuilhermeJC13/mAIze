# MazeIA-

How to play the game:

1. Clone the repo
   ```sh
   git clone https://github.com/TailUFPB/mAIze.git
   ```
2. Install packages
   ```sh
   pip install requirements.txt
   ```
3. Run the game file (Windows)
   ```sh
   python player_game\env\game.py
   ```
   (Linux)
   ```sh
   python player_game/env/game.py
   ```
